Description

The app measures user's sun exposure by using outdoor UV index obtained through OpenWeatherMap API and user's indoor/outdoor status.

On the start screen of the App, real-time UV index, current geolocation coordinates and number of satellites are displayed. The app also collects GPS signal, Cell signal and Wifi signal in the backend with built-in sensors. All data are collectd once per minute.

The menu page allows the user to view UV diagram that shows daily, weekly and monthly UV exposure. User can also go to questionnaire to test their understanding of UV exposure.

When service of the app is stopped by the user, all data collected will be written to csv file and stored in the internal storage of the phone.
